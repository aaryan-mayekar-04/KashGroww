# 🚀 KashGroww – Robo Advisory Investment Platform

KashGroww is a **smart robo-advisory web application** designed to help investors build **personalized, diversified, and goal-oriented investment plans**.
It simplifies investment decision-making by combining **user profiling, strategic asset allocation, and systematic planning** into one seamless platform.

---

## 📌 Key Features

* 🎯 **Personalized Investment Plans**
  Generates customized portfolios based on user preferences, goals, and risk appetite.

* 📊 **Diversified Asset Allocation**
  Recommends a balanced mix of asset classes to optimize risk and returns.

* 🔄 **Systematic Investment Strategy**
  Encourages disciplined investing through structured planning.

* 🔐 **User Authentication System**
  Secure login & signup with OTP/email verification.

* 🧠 **Smart Recommendation Engine**
  Uses logic-driven analysis to suggest optimal investment strategies.

* 📈 **Interactive Dashboard**
  Displays portfolio insights, selected assets, and planning summary.

## ⚙️ How It Works

1. User signs up / logs in securely
2. Inputs:

   * Financial goals
   * Risk tolerance
   * Investment preferences
3. System processes inputs
4. Generates:

   * Diversified portfolio
   * Strategic allocation
   * Personalized investment plan

## 🚀 Installation & Setup

1. **Clone the repository**

```bash
git clone https://github.com/aaryan-mayekar-04/KashGroww.git
cd KashGroww
```

2. **Create virtual environment**

```bash
python -m venv venv
source venv/bin/activate   # (Linux/Mac)
venv\Scripts\activate      # (Windows)
```

3. **Install dependencies**

```bash
pip install -r requirements.txt
```

4. **Configure Database**

* Set up MySQL
* Update database credentials in config file

5. **Run the application**

```bash
streamlit run app.py
```
